Why Loqi

Built for people gluing an LLM SDK to an HTTP lib to a JSON lib.

You like Python's readability but want a single fast binary. Loqi puts the whole AI toolchain inside the language.

AI is in the language

ai(prompt) is a first-class call — not a bolted-on package. It speaks to the Anthropic Messages API, reads ANTHROPIC_API_KEY, and defaults to a sensible model you can override per call.

Batteries included

One binary, no dependency hell. JSON, HTTP, vector similarity, file and env I/O, plus 25+ core built-ins ship inside the runtime. Nothing to pip install, nothing to vendor.

Reads like Python, runs native

Clean, readable syntax with no semicolons and string interpolation — compiled to bytecode for a stack VM. One fast native binary on Apple Silicon, and getting faster each iteration.

ai(prompt)

A first-class LLM call to the Anthropic Messages API. Reads ANTHROPIC_API_KEY; pass a model as a second arg or via LOQI_AI_MODEL.

let poem = ai("Scrivi un haiku sul codice pulito")
print(poem)
json

json.parse(str) and json.stringify(value) — a built-in codec between JSON text and native Loqi values.

let cfg = json.parse(`{"name": "Ada"}`)
print(cfg.name)
http

http.get(url) and http.post(url, body, content_type?) — a built-in HTTP client returning the response body as a string.

let body = http.get("https://api.github.com")
print(len(body))
similarity(a, b)

Cosine similarity over numeric vectors — the primitive behind semantic search, ranking and clustering.

let s = similarity(a, b)
print(s)  # a float in [-1, 1]
read / write

read(path) returns a whole file as a string; write(path, content) writes one back. Simple, whole-file I/O.

write("out.txt", "ciao")
print(read("out.txt"))
env(name)

Read environment variables straight from the runtime — keys, config and feature flags, no wrapper library.

let key = env("ANTHROPIC_API_KEY")
print(type(key))
Honest about speed

Python-class speed, often faster on loops.

Measured on Apple Silicon with process_time. On fib(30) Loqi is on par with CPython 3.13. On a tight loop to 50M, Loqi is about 1.4× faster than CPython — and roughly 5× faster than Loqi's own first tree-walking interpreter.

The honest part: Node/V8 runs fib(30) faster (0.017s) because it JITs. Loqi does not JIT yet. We do not claim to beat Go, Rust, C or Node — just a single compiled binary with Python-class speed that gets faster each iteration.

By example

Real Loqi, top to bottom.

Recursion, closures, AI-powered extraction, and verbatim raw strings for JSON — every snippet below runs as-is.

fib.lq recursion 
fn fib(n) {
  if n < 2 { return n }
  return fib(n - 1) + fib(n - 2)
}
print("fib(10) = {fib(10)}")
contatore.lq closures 
fn contatore() {
  let n = 0
  return fn() { n = n + 1; return n }
}
let next = contatore()
print(next(), next(), next())   # 1 2 3

Frequently asked questions

What is Loqi?

Loqi is an AI-first programming language: calling an LLM with ai(), parsing JSON, making HTTP requests, and computing vector similarity are built into the language and its runtime, not separate packages. It's implemented in dependency-free C as a bytecode VM and is on par with CPython on recursion, ~1.4× faster on tight loops.

What makes it AI-first?

ai(prompt), json.parse/json.stringify, http.get/http.post, similarity(), env, read and write are built in. You write an AI program with no HTTP client, no JSON library, no LLM SDK, and no virtualenv or node_modules to install.

How fast is it?

On fib(30), Loqi runs in ~0.097s versus CPython 3.13's ~0.094s (on par); on a 50-million-iteration loop it's ~1.4× faster than CPython. It doesn't JIT yet, so V8 wins micro-benchmarks — that's the next speed iteration.

How do I install it?

Clone github.com/ferdinandobons/loqi and run ./scripts/build.sh (needs only the macOS Xcode Command Line Tools) to get ./build/loqi.

Loqi

The AI-first language — simple to read, fast on Apple Silicon. A brand-new language with its own grammar, parser and stack VM, built in dependency-free C11.

Honest about limitations: no JIT yet (so V8 wins micro-benchmarks), no GC yet (the arena is freed at exit — fine for scripts, GC is planned), single-file modules, Apple-Silicon-first, and http/ai shell out to curl today. It's all on the public roadmap.
